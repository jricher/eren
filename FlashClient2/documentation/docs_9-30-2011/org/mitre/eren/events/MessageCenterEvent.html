Classpublic class MessageCenterEvent
InheritanceMessageCenterEvent flash.events.Event

Custom event class for message center related events. This is a super-typed event; all events are fired as type MESSAGE_CENTER_EVENT, but once the event is recieved e._type will give the event type.
