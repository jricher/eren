Classpublic class MessageCenterManager
InheritanceMessageCenterManager flash.display.MovieClip

The MessageCenterManager manages the Message Center (instantiated on Stage). The Message Center includes a MssgQueue of incoming messages, a window for viewing and replying to open messages, a chat window (stub, not implemented), and a resource vizualization panel.
