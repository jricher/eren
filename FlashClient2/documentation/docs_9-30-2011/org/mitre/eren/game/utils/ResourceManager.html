initFacilities()method 
public function initFacilities(a:Vector.<facility>):void

Initialize the facilities list. The manager must know about all facilities at the start of the game, whether they are in use or not. Therefore there is no public "addFacility(f)" method. This method should only be called once the RM has been created on the stage. Calling it before the RM has been added to the display list will result in error.

Parameters

a:Vector.<facility> — an array of Facility objects to manage
