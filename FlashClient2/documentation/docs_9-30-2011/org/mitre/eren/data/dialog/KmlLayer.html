Classpublic class KmlLayer
InheritanceKmlLayer flash.events.EventDispatcher

A KmlLayer uses the KmlLoader from the distriqt library to load kml files from a given set of urls, convert those files into display objects, and insert those display objects into a Google Maps IPane. This IPane can then be directly added to the map instance.
