Packageorg.mitre.eren.data.actions
Classpublic class Input
InheritanceInput Object

Input object, representing an "act:input" node from the Actions File. An input always has a type, name, fromId (id of the npc the question text should appear to come from), and questionText. Min, max, and the set of options are optional.
