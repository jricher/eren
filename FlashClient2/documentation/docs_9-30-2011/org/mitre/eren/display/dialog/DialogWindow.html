setPeopleDictionary()method 
public function setPeopleDictionary(p:Dictionary):void

Set the people dictionary. The people dictionary is a Dictionary object containing Person objects representing all of the NPCs registered in the Scenario File. Person references are used to populate sender information from userMessages, which only contain the ID of the sender NPC.

Parameters

p:Dictionary — an npc id-indexed dictionary of Person objects
