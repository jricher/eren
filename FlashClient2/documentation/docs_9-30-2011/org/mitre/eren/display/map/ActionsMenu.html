Classpublic class ActionsMenu
InheritanceActionsMenu flash.display.MovieClip

An ActionsMenu contains a set of MenuOptions. Each menu is linked to a specific facility, so it has a reference to the facility's id. When the ActionsFile is parsed, the ResourceManager creates an ActionsMenu for each facility (if it has actions).
