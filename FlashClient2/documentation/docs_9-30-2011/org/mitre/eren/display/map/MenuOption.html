Classpublic class MenuOption
InheritanceMenuOption flash.display.MovieClip

A MenuOption represents an action that a player may take on a particular facility. MenuOptions are contained inside an ActionsMenu, and each MenuOption contains a reference to the Action object it represents. When a MenuOption is enabled, it is displayed in blue with white text. If the option is disabled, it is displayed greyed out and is not clickable.
